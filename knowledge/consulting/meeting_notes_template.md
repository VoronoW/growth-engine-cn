# Growth Consulting Meeting Notes Template

适用于：
跨境增长咨询项目 / 增长加速器项目 / 战略咨询项目会议记录。

目标：
帮助咨询团队系统记录项目讨论、决策与执行任务，
确保增长项目可以持续推进并形成可追踪的执行体系。

# 2. Meeting Background

说明本次会议召开背景。

需要回答：

- 当前项目所处阶段
- 本次会议召开原因
- 本次会议需要解决的问题
- 与上一阶段工作的关系

示例：

当前项目处于 **数据基建阶段**。  
企业已完成多平台数据导出与初步清洗。  

本次会议目标是：

- 评估当前数据归因体系
- 确认技术排雷优先级
- 制定下一阶段执行计划。

# Appendix: Common Consulting Meeting Types

建议在增长咨询项目中固定以下会议体系：

| Meeting Type | Frequency | Purpose |
|----|----|----|
| Project Kickoff | Project Start | Align goals |
| Weekly Execution Meeting | Weekly | Track tasks |
| Data Review Meeting | Biweekly | Analyze metrics |
| Growth Strategy Meeting | Month 2 | Confirm experiments |
| Phase Review Meeting | Monthly | Evaluate results |
| Project Closing Meeting | Project End | Deliver system |
