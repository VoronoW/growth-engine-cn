# Cross-Border Growth BI Dashboard Framework

本文件定义跨境企业增长 BI 仪表盘（Growth Dashboard）的标准结构。  
适用于跨境电商、品牌出海、独立站、TikTok Shop、Amazon 等企业的数据指挥系统。

BI Dashboard 的目标是帮助企业实现：

- 实时监控增长状态
- 快速定位增长问题
- 支持广告与运营决策
- 提供 CEO 级增长视角

# 使用说明

本 BI Dashboard Framework 可用于：

- 构建企业增长数据指挥中心
- 设计 Looker / PowerBI / Metabase 仪表盘
- 搭建跨境增长数据中台
- 支持 CEO / Growth Lead 决策
- 监控跨境业务增长健康度

推荐在 BI 系统中建立以下三层结构：

1. Executive Dashboard（CEO增长驾驶舱）
2. Marketing Dashboard（广告与流量分析）
3. Product & User Dashboard（产品与用户分析）

该结构可以帮助跨境企业实现从“买量公司”向“增长系统公司”的升级。
