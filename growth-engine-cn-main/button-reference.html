01 / VARIANT
六种语义变体
default 是主要行动；outline / secondary 是次要操作的两种密度； ghost 用于工具栏或内联；destructive 用软色调承载危险行动；link 退回到排印。
default
bg-primary · text-primary-foreground
outline
border + bg-background
secondary
bg-secondary · text-secondary-foreground
ghost
transparent · hover:bg-muted
destructive
bg-destructive/10 · text-destructive
link
text-primary · underline-offset-4
02 / SIZE
四挡文字尺寸
高度 24 / 28 / 32 / 36 → xs / sm / default / lg。xs 与 sm 使用较小 radius（10px / 12px）， 让密度高的工具栏看起来更紧实；default 与 lg 保持 8px radius 的标准节奏。
xs · h-6 (24px)
text-xs · radius 10
sm · h-7 (28px)
text-0.8rem · radius 12
default · h-8 (32px)
text-sm · radius 8
lg · h-9 (36px)
text-sm · radius 8
03 / ICON
纯图标按钮
正方形尺寸与文字版本对齐。icon-xs / icon-sm 使用较小 radius， 让它们在文本输入框的 addon 或高密度工具栏里视觉更整齐。
icon-xs · size-6
24 × 24 · radius 10
icon-sm · size-7
28 × 28 · radius 12
icon · size-8
32 × 32 · radius 8
icon-lg · size-9
36 × 36 · radius 8
04 / WITH ICON
文字 + 图标
用 data-icon="inline-start" 或 inline-end" 触发自动的 padding 收缩，让图标侧的视觉重量比纯字版本收紧一档，避免两侧过度对称。
inline-start — 前置图标
pr-2.5 → pl-2 on icon side
inline-end — 后置图标
pl-2.5 → pr-2 on icon side
05 / LOADING
加载状态
用 Spinner + disabled 组合，而不是内置的 isLoading prop。 图标位使用 data-icon，保持 padding 节奏一致。

opacity-50 · pointer-events-none
outline · pending
border 保持稳定
icon · pending

06 / ROUNDED
全圆角
加 rounded-full 即可，不做单独 variant。用于 tag-like 操作、navbar CTA， 或需要视觉减压的区域。

07 / GROUP
按钮组
相邻按钮共享边界：首尾保留外侧 radius，中间变直角，负间距合并边框。 用于分段控件、编辑器工具栏、分页控制。
segmented · outline
aria-expanded 表示选中态
toolbar · ghost + icon
相邻 margin-left:-1px 合并边
08 / DARK
深色表面上的行为反转
ZAPEX 首页的 hero / 核心模块使用深底。按钮 tokens 原样继承， 只是主色从黑变白 — 保证在 deck / brochure 场景下读起来是一个体系。

10 / SLOT · RTL
asChild 与双向文字
asChild 把任意元素（通常是 <a>）套上按钮外观，不新增 DOM 层。 RTL 下图标位自动镜像（inline-start 仍是语义上的前置）。
