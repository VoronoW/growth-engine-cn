## 三、飞书建表操作步骤

1. 打开飞书，新建一个**多维表格**，命名为「增长引擎验证舱 · 数据表」
2. 在该多维表格中依次新建 **9 张数据表**（对应上方 A1–B6）
3. 按照上方字段映射表为每张表添加列，字段类型严格对照表格
4. 单选字段需提前配置好所有枚举选项
5. 每张表建好后，点击右上角「…」→「API」→ 复制 `table_id` 填入 `TABLE_MAP`
6. 从多维表格 URL 中复制 `app_token` 填入 `BITABLE_APP_TOKEN`

在 growth-engine-cn repo 中完成以下操作：

1. 将 feishu-config.js 放入 /js/ 目录

   <script src="/js/feishu-config.js"></script>

3. 将 growth-forms.html 中原有的 BITABLE_CONFIG 对象（约第458行）
   替换为以下代码，使其读取 feishu-config.js 的全局变量：

   var BITABLE_CONFIG = {
     app_token: BITABLE_APP_TOKEN,
     tables: TABLE_MAP
   };

4. 在 submitToFeishu() 函数中，提交数据前用 FIELD_MAP 转换字段名：
   将 HTML 表头映射为飞书字段名，避免字段不匹配导致写入失败。
   示例：
   var map = FIELD_MAP[panelId] || {};
   var mappedFields = {};
   Object.keys(fields).forEach(function(k) {
     var feishuKey = map[k] || k;
     mappedFields[feishuKey] = fields[k];
   });
   // 用 mappedFields 替代 fields 提交

5. 填写 feishu-config.js 中所有 tbl_xxx 为实际 table_id 后部署
